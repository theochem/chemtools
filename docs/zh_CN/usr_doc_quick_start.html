快速入门¶

ChemTools 是一款用来理解、分析计算化学计算结果的软件库。

在使用ChemTools之前，您需要先使用 HORTON 或者别的计算化学软件对您感兴趣的体系进行电子结构计算并得到输出文件，再利用ChemTools对输出文件进行后处理和相关分析。目前，ChemTools支持的文件格式包括

如果您想要尝试所有的教程，请先确保您的电脑上已正确安装了ChemTools及其必需的软件库。您可以参照 安装指南 页面，根据您电脑的系统，选择相对应的安装方法。

如何使用ChemTools¶

ChemTools作为一款用于量子化学后处理的Python软件库。如同其他Python软件库一样，您可以在Python脚本中直接导入（import）使用，但是这需要一定的Python编程知识。如果您对编程或者Python不是特别熟悉的话，ChemTools也内置了一系列的脚本，并提供命令行形式的交互界面。

Python软件库用法¶

如果您有一定的编程经验，我们强烈建议您将ChemTools当作一款独立的Python软件库来使用。这使您能够充分使用ChemTools的所有特性，同时也能更好的利用现有工具来辅助实验及研究。

下面就是使用ChemTools作为Python软件库所写的一小段代码。您可以用这段代码作为模版，编写您自己的脚本来进行相关的计算。

#!/usr/bin/env python

# import ChemTools library
import chemtools

# print version of ChemTools library
print chemtools.__version__

通常，这些代码被保存成后缀名为``.py``的文件中，如例中的``script.py``。您可以使用``python script.py``来运行脚本，也可以将脚本设定为可执行文件再运行，如例中所示的方法。

$ chmod +x script.py
$ ./script.py

您可以跳转至 新手教程 查看ChemTools各式各样的功能或 样例图库 查看相关实例展示及其代码。

Python脚本用法¶

ChemTools内置的脚本让其不依赖其他程序就能在命令行中顺利运行。我们试图在脚本模式中增添尽可能多的功能，但是仍有部分特性不能实现。这些内置的脚本都有类似于 chemtools-*.py 的文件名。您可以使用 --help 或是 -h 选项来查看这些脚本的功能和用法，例如：

$ chemtools-nci.py --help
